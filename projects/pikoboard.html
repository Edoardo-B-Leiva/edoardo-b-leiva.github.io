PiKoboard


The PiKoboard is a project that involves the use of a Raspberry Pi Pico as a substitutive of an MCU inside of some custom keyboards.

This brings:

  • Hard to brick and easibly installabe and updateable firmware for custom keyboards.
  • No need for extra tools such as programs and devices unlike normal EEPROMs and MCUs
  • Easibly customizable firmwares with custom features (RGB support, macros, extra function keys(vol+ ,vol-, SCRL lock))
